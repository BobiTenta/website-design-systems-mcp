# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-09

### Added

- Initial release
- `extract_design_system` tool - Complete design system extraction to skill.md
- `validate_url` tool - URL accessibility check
- `get_site_colors` tool - Color palette extraction
- `get_site_typography` tool - Typography system extraction
- Anti-blocking HTTP client with User-Agent rotation
- Cheerio-based HTML parsing
- CSS extraction from inline styles, style tags, and external stylesheets
- Color normalization (hex, rgb, rgba, hsl, hsla, named colors)
- Typography extraction (fonts, type scale, weights, line heights)
- Spacing system detection
- Border radius and shadow extraction
- Layout and breakpoint analysis
- Brand identity extraction (logo, favicon, meta tags)
- Comprehensive skill.md generation with YAML frontmatter
